Webd | A lightweight and portable web-based file sharing server software.

  • It's extremely lightweight, less than 90 kB (with a webserver built in), and requires no other dependencies.
  • No complex setup or installation procedures, simply unzip/untar and run, and then a local folder got showed on web.
  • Flexible permission control: Accessing, listing, uploading, renaming/moving, show hidden files, all can controlled independently.
  • Supporting multiple platforms, including Windows(Win7 and above version), Linux, OpenWrt and Armbian. With "adb shell", it can even be installed on Android.
  • Media Streaming: Directly play videos without downloading.
  • Almost Ads free: Just an inconspicuous single line of text Ads that might appear at the bottom of the file list.

Quick Start:

  1. Download the latest webd packages for specific OS.
  2. Unzip/Untar it.
  3. Run in command line:

    Linux
    ./webd -w /home/user/share -u rlum:user:pass
    Windows
    webd.exe -w D:\sharedata -u rlum:user:pass
    The folder /home/user/share or D:\sharedata will be published/shared on web.
    Then open http://127.0.0.1:9212 in browser on the same computer to check if it work.
    On other devices just open a URL like http://192.168.12.34:9212 to use webd.
    Note: The "192.168.12.34" must be changed to the actual IP address of machine that running webd.
    Without login, webd only allows the "listing" and "accessing" permissions by default setting.
    After login with "user" and "pass", webd will give extra permissions: uploading, renaming/moving, creating new folder.
    "user" and "pass" should be changed to other values for safety.

  4. Use Ctrl + C in command line window to stop webd.
